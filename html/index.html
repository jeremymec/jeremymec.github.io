Things I've Worked On

Web App

Hangman on Rails

Rails 5 Javascript Bootstrap Droplet

Hangman on Rails is a full stack web application written in Ruby on Rails, with bootstrap for styling. It's a simple and clean implementation of the classic hangman game, allowing you to create games than can easily be shared with your friends through a persistent URL. I decided to create this game when my friends and I could not find a website to play online hangman which wasn't riddled with adds, and it turned into an extendable passion project that I plan to add more features to.

Game

Insanity Pong

Java Processing

Insanity Pong was created by myself and a friend for Kiwijam 2019. Our idea was simple - to take Pong, and dial it up to 11. It has power-ups, constant random spawning, balls that collide into groups that do extra damage, and healthbars which turn it into a war of attrition. It was written in two days using a Processing library for Java to combine the ease of use of processing with the advanced data structures and algorithms that Java provides. It was great fun to work on!

Check It Out Github

Mobile App

Flat Manager

Ionic Angular HTML Firebase

The Flat Manager app started off as a university assignment, but I began to extend on it when I realised how useful a fully developed version would be in my life. It allows you to create a flat and invite your friends, providing things like a shared shopping list and a newsfeed. It's built on ionic running on top of angular, meaning it runs on the web, iOS, or android. The app uses firebase for its authentication and the rest of the backend is provided by a Rails API I created.
